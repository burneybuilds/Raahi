<h1 align="center">Raahi</h1>


Raahi is a Python route finder project that uses OpenStreetMap data to find routes between locations and display them on an interactive map.

I built this project to understand how route finding applications work behind the scenes. The goal of this project is to learn how place names are converted into coordinates and how routes are generated between two locations.

Raahi takes a location name, converts it into latitude and longitude coordinates, and uses routing algorithms to find routes between places. The generated routes are displayed on an interactive map using OpenStreetMap data.

This project is built with Python and uses Requests for API communication, Folium for map visualization, OSMnx for working with road networks, and NetworkX for route generation and graph algorithms.
